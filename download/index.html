Minimum System Recommendations

  • 1.5GB free memory
  • 1GB free swap

I want to install API Umbrella on

  • Ubuntu 12.04
  • Ubuntu 14.04
  • Debian 7
  • Enterprise Linux 6 (CentOS/RedHat/Oracle/Scientific Linux)
  • Enterprise Linux 7 (CentOS/RedHat/Oracle/Scientific Linux)

Download and install the package:

wget https://developer.nrel.gov/downloads/api-umbrella/ubuntu/12.04/api-umbrella_0.8.0-1_amd64.deb
sudo apt-get install gcc
sudo dpkg -i api-umbrella_0.8.0-1_amd64.deb

Open the configuration file for editing:

sudo -e /etc/api-umbrella/api-umbrella.yml

In the config file, define an e-mail address for your first admin account:

web:
  admin:
    initial_superusers:
      - your.email@example.com

Start API Umbrella:

sudo /etc/init.d/api-umbrella start

Browse to your server's hostname. You should land on the default homepage:

Congrats! You're now up and running with API Umbrella. There are a variety of things you can do to start using the platform. For a quick tutorial, see getting started.

Problems? Open an issue.
