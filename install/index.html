Install

Minimum System Recommendations

  • 1.5GB free memory

I want to install API Umbrella on

  • Docker
  • Ubuntu 12.04 (Precise)
  • Ubuntu 14.04 (Trusty)
  • Ubuntu 16.04 (Xenial)
  • Debian 7 (Wheezy)
  • Debian 8 (Jessie)
  • Enterprise Linux 6 (CentOS/RedHat/Oracle/Scientific Linux)
  • Enterprise Linux 7 (CentOS/RedHat/Oracle/Scientific Linux)
  • Other

In this simple example, the API Umbrella configuration can be stored in the config/api-umbrella.yml file on host machine.

$ mkdir config && touch config/api-umbrella.yml

Open the configuration file for editing. In the config file, define an e-mail address for your first admin account:

web:
  admin:
    initial_superusers:
      - your.email@example.com

Start API Umbrella:

$ docker run -d --name=api-umbrella -p 80:80 -p 443:443 -v "$(pwd)/config":/etc/api-umbrella nrel/api-umbrella

Download and install the package:

echo "deb https://dl.bintray.com/nrel/api-umbrella-ubuntu precise main" | sudo tee /etc/apt/sources.list.d/api-umbrella.list
sudo apt-get update
sudo apt-get install api-umbrella

Download and install the package:

echo "deb https://dl.bintray.com/nrel/api-umbrella-ubuntu trusty main" | sudo tee /etc/apt/sources.list.d/api-umbrella.list
sudo apt-get update
sudo apt-get install api-umbrella

Download and install the package:

echo "deb https://dl.bintray.com/nrel/api-umbrella-ubuntu xenial main" | sudo tee /etc/apt/sources.list.d/api-umbrella.list
sudo apt-get update
sudo apt-get install api-umbrella

Download and install the package:

Download and install the package:

curl https://bintray.com/nrel/api-umbrella-el7/rpm | sudo tee /etc/yum.repos.d/api-umbrella.repo
sudo yum install api-umbrella

We build binary packages for a variety of Linux distributions. These binary packages are the recommended way to install API Umbrella. However, if binary packages aren't available, you can also compile from source (also, please let us know if you'd like to see binaries for other operating systems):

$ curl -OLJ https://github.com/NREL/api-umbrella/archive/v0.13.0.tar.gz
$ tar -xvf api-umbrella-0.13.0.tar.gz
$ cd api-umbrella-0.13.0
$ make
$ sudo make install

Open the configuration file for editing:

sudo -e /etc/api-umbrella/api-umbrella.yml

In the config file, define an e-mail address for your first admin account:

web:
  admin:
    initial_superusers:
      - your.email@example.com

Start API Umbrella:

sudo /etc/init.d/api-umbrella start
